   6. AI evaluates each answer

   8. AI provides closing message

QUICK START COMMANDS
========================================================

1. Setup Backend:
   cd backend
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   echo 'GOOGLE_API_KEY=your_key' > .env

2. Setup Frontend:
   cd frontend
   npm install

3. Run Application:
   cd ..
   ./start.sh

4. Open Browser:
   http://localhost:3000
